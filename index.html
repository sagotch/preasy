PREASY

Easy HTML presentation

  • One markdown file per slide
  • Customized with CSS / JavaScript
  • Keyboard support for slide switching
    • → key moves forward
    • ← key moves backward

Organise your presentation folder

  • css/: CSS file(s) to link with your HTML page
  • js/: JavaScript file(s) to link with your HTML page
  • res/: Ressources used by slides (e.g. pictures)
  • slides/: One markdown file per slide
.:
css  res  slides

./css:
font-league-gothic.css preasy.css

./res:
font-league-gothic  preasy-screenshot.png

./res/font-league-gothic:
leaguegothic-regular-webfont.eot  leaguegothic-regular-webfont.ttf
leaguegothic-regular-webfont.svg  leaguegothic-regular-webfont.woff

./slides:
01.md  02.md  03.md  04.md  05.md  06.md

Build you presentation

preasy build

Open _build/index.html
